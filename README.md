<h1 align="center">My Portfolio</h1>

## 🚀 About This Project
This repository contains the **portfolio website** I built using **Laravel 10** for the backend and **Vue 3 + Tailwind CSS** for the frontend.  
It showcases my projects with images, descriptions, tech stacks, and links to GitHub/live demos.

---

## 🧰 Tech Stack
- **Backend:** Laravel 10, Sanctum API
- **Frontend:** Vue 3, Tailwind CSS, Axios
- **Database:** MySQL
- **Deployment:** Laravel Forge / Your Hosting

---

## ✨ Features
- Dynamic **Admin Panel** (CRUD for projects)
- Responsive & modern **UI**
- Hero section with animated particles + project slider
- Project filtering & pagination
- Secure auth using **Laravel Sanctum**
- Dark & light mode support
